# Moonlight Web Studio（ポートフォリオサイト）

## 概要
  ポートフォリオや実績をまとめるサイトを作りました。
  Canva AIでデザインや構成の指示を出して生成されたコードをベースに、不要なコードを削除し、構造やクラス名などを整理して自分の望むサイト構成に合わせて再構築しています。

## 工夫した点

 * Canva AIで生成されたコードをそのまま使用するのではなく、HTML構造やスタイルを読み解きながら整理しました
 * 不要な要素やスタイルを削除し、保守しやすいクラス名を付け直して管理しやすい構造に再設計しました
 * セクションごとにCSSを整理し、アニメーションや余白を見やすい構成に改善しました
 * JavaScriptの動作を確認しながら必要な機能のみ残し、シンプルな構成になるよう調整しました
 * ContactフォームはEmailJSを利用してフロントエンドのみで送信可能にしました
